Requirements:
1. Download and Install Eclipse IDE for Java Developers
2. Download and Install Java JDK
3. Download and Install Java 7
4. Import the Project on Eclipse
5. Google account with access to Google Drive

Instructions to run:
1. All the files that are to be executed are present under DemoDrive/src/test/java/DS3/ 
2. Run Controller.java and Peer.java on one machine with the IP address in main function of Peer.java set to the "localhost"
3. Run Peer.java on other machines with the IP address in main function of Peer.java set to the Controller's IP address
4. Now the peers are running, follow the workflow in the project report to test the project
